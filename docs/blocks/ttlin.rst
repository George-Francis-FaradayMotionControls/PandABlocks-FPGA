TTLIN - TTL Input
=================

The TTLIN block handles the signals from the TTL Input connectors

Parameters
----------

=============== === ======= ===================================================
Name            Dir Type    Description
=============== === ======= ===================================================
TERM            R/W Enum    | 0 - Sets input termination to High-Z
                            | 1 - Sets input termination to 50-Ohm
VAL             Out Bit     TTL Input value
=============== === ======= ===================================================
